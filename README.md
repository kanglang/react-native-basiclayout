# react-native-basiclayout

## Usage 使用
```javascript
import Basiclayout from 'react-native-basiclayout';

// TODO: What to do with the module?
<Basiclayout>
    <Text>这里是子元素</Text>
</Basiclayout>
;
```
## Basiclayout 基础容器组件

| 属性                 | 功能                    | 类型             | 默认值     |
| ------------------- | -----------------------  | -------------   | ---------  |
| isScroll            | 容器是否需要滚动          | boolean         | false      |
| style               | 容器样式自定义            | style           | --         |
| children            | 容器子元素                | React element   |  --        |
